# rifactor

AWS Reserved Instance Optimization

## Prerequisites

- GHC 7.10
- cabal-install 1.22

## Build

[![Build Status](https://travis-ci.org/dysinger/rifactor.png)](https://travis-ci.org/dysinger/rifactor)

    make

## Test

    make test

## Install

    install ./.cabal-install/bin/rifactor /usr/local/bin

## Run

    rifactor --help
